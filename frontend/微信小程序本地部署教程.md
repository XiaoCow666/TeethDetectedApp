# 🦷 口腔 AI 筛查系统 - 本地部署指南

本指南旨在帮助开发者在本地环境中快速启动 AI 后端及微信小程序前端。

---

## 一、 环境准备
* **系统**: Windows 10/11 (建议配备 NVIDIA GPU)
* **软件**: 
    * Python 3.8+
    * 微信开发者工具 (Stable Build)
    * [可选] cpolar (用于内网穿透)

---

## 二、 后端服务启动 (FastAPI)

1. **安装依赖**:
   ```bash
   pip install -r requirements.txt
2. **放置模型**
    
    确保 best.pt 或 best.onnx 已放置在项目的 models/ 或 weights/ 目录下。
3. **启动服务**

    打开终端，切换到项目根目录，运行：
    ```bash
    # --host 0.0.0.0 允许局域网内其他设备(如手机)访问
    uvicorn api.main:app --host 0.0.0.0 --port 8000 --reload
   
  验证: 浏览器访问 http://127.0.0.1:8000/docs 查看 API 文档。
  
## 三、 前端小程序配置

**项目导入：**

1.打开微信开发者工具，选择“导入项目”，目录指向 frontend 文件夹。

2.AppID 使用你的测试号或正式号。

**本地设置 (关键)：**

1.点击界面右上角 “详情” -> “本地设置”。

2.勾选 “不校验合法域名、web-view、TLS版本以及HTTPS证书”。

3.接口地址配置：
在 pages/camera/camera.js 中配置 BASE_URL：

模拟器：http://127.0.0.1:8000


真机调试：http://你的电脑局域网IP:8000 

## 四、 真机调试步骤

**开启调试模式：**

1.在手机上预览小程序后，点击右上角 “...” -> “开发调试” -> “打开调试”。

2.看到悬浮的绿色 vConsole 按钮后重启小程序。

3.网络检查：

4.确保手机浏览器能访问 http://电脑IP:8000/。如果无法访问，请检查 Windows 防火墙设置。
