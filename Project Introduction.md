# 🦷 AI 口腔健康管家 (Oral AI) - 项目介绍文档

## 1. 项目简介 (Project Overview)
**“AI 口腔健康管家”** 是一款基于深度学习（YOLOv8）与前后端分离架构的智能移动端辅助筛查工具。本项目致力于打破传统口腔医疗的物理壁垒，将专业的“全视野临床视觉诊断能力”下放至用户的智能手机中。

用户仅需通过微信小程序拍摄口腔照片，系统即可在 **毫秒级 (约 48ms)** 内完成复杂的影像分析，精准框选病变区域，并依托内置的**医疗规则引擎 (Rule Engine)** 自动生成包含健康评分、严重级别及就医建议的结构化报告，实现口腔疾病的“早发现、早干预、早治疗”。

## 2. 核心功能与 9 大病变识别矩阵 (Core Capabilities)
本项目的 AI 核心大脑采用经过大规模专业医疗数据训练的 9 分类深度学习模型。有别于市面上仅能识别“蛀牙”的单一模型，本系统实现了跨牙体、牙周及黏膜的**全科综合评估**。

系统支持精准定位并分类以下 **9 种** 口腔状态：

| 标签 ID | 英文分类名 | 中文病变名称 | 临床筛查价值 / 严重程度 |
| :--- | :--- | :--- | :--- |
| `0` | **CALCULUS** | 牙结石 | **中度风险**：牙周炎及牙齿松动的主要诱因 |
| `1` | **Gingivitis** | 牙龈炎 | **中度风险**：提示牙龈红肿发炎，需改善卫生习惯 |
| `2` | **Lichen** | 疑似扁平苔藓 | **高风险**：精准捕获黏膜网状/斑块异常，预防癌变风险 |
| `3` | **caries** | 疑似龋齿 (蛀牙) | **高风险**：识别牙面龋坏痕迹，提示及时修补 |
| `4` | **decaycavity**| 严重深龋洞 | **极高风险**：病变深达牙本质或牙髓，预警根管治疗 |
| `5` | **earlydecay** | 早期脱矿/浅龋 | **低风险**：突破肉眼局限，捕获亚临床微小脱矿变色 |
| `6` | **healthytooth**| 健康牙齿 | **健康指标**：作为基准对照，系统自动过滤不作扣分 |
| `7` | **missing** | 牙齿缺失 | **中度风险**：预警邻牙倾斜与咬合紊乱，建议义齿修复 |
| `8` | **plaque** | 牙菌斑沉积 | **低风险**：评估日常刷牙死角与清洁度 |

## 3. 系统架构与技术亮点 (Technical Highlights)

本项目采用轻量级、高并发的现代前后端分离架构设计：

* **前端 (微信小程序)**:
    * **极致 UX 设计**：采用具有医疗专业感的 `#00BFA5`（医疗绿）为主色调，结合现代化的毛玻璃与卡片式 UI 设计。
    * **动态靶向渲染**：接收后端返回的归一化坐标 (`xyxyn`)，利用 CSS 绝对定位，在原图上**动态绘制高亮病变红框与标签**，实现“眼见为实”的震撼交互。
* **后端 (FastAPI + Python)**:
    * **极速推理**：采用 FastAPI 异步框架，结合导出优化的 YOLO 模型，实现高并发下的毫秒级影像推理。
    * **首创医疗规则引擎 (RuleEngine)**：不仅输出 AI 标签，更将医学逻辑代码化。引擎会根据病变类型计算动态健康分（`health_score`），并为用户匹配千人千面的中文医疗建议与风险评级。
* **物联网生态拓展 (IoT Ready)**:
    * 底层架构已预留视频流处理接口，未来可无缝接入百元级 **Wi-Fi 智能口腔内窥镜**，打破手机摄像头物理限制，实现软硬一体化的医疗级创新。

## 4. 商业模式与社会价值 (Business Value & Impact)
* **普惠医疗 (Social Impact)**：为医疗资源匮乏地区提供零门槛的家庭牙医服务，将国民口腔健康管理关口前移。
* **O2O 精准导流 (Commercialization)**：通过 AI 报告直击用户痛点，系统底层已设计“一键预约附近牙医”的商业闭环接口，可作为线下高端口腔诊所的超级精准获客漏斗。
* **数据追踪 (Data Tracking)**：通过生成个人/家庭口腔数字档案，量化用户的卫生改善趋势，大幅提升应用留存率。
