 # 口腔 AI 数据标注规范 (Label Guidelines)

> **一致性是核心**。模型训练质量完全取决于标注的统一性。请团队全员严格遵守本规范。

## 1. 工具与格式
*   **推荐软件**: LabelStudio (团队协作) 或 LabelImg (单机)。
*   **数据格式**: YOLO (txt文件) - 包含检测框与分割多边形。

## 3. 标注细则

### 通用原则
*   **紧密贴合**：检测框/多边形必须紧贴目标边缘，不要包含多余背景。
*   **遮挡处理**：如果目标被嘴唇、舌头或器械遮挡超过 **50%**，则**不要标注**。
*   **存疑不标**：如果你只有 70% 以下的把握，**不要强行标注**。宁可漏检，不要误导模型。

### 分类细则
*   **牙齿 (`tooth`)**：
    *   仅标注恒牙/乳牙，除非特殊说明，不标注假牙/种植牙。
    *   **分割任务**：沿牙齿可见边缘描点，忽略被牙龈覆盖的部分。
*   **龋齿 (`caries_*`)**：
    *   圈出**整个变色区域**，而不仅仅是中心最黑的点。
*   **牙龈炎 (`gingivitis_*`)**：
    *   标注具体的**发红/红肿区域**，通常是沿牙颈部的月牙形区域。

## 4. 质量自检清单
导出数据前，请检查：
1.  [ ] 所有清晰可见的牙齿都标了吗？(召回率)
2.  [ ] 框是否足够紧凑？(精确度)
3.  [ ] 是否有误触产生的极小框（Ghost Boxes）？
4.  [ ] 类别 ID 是否选择正确？
